🎬 Osei Movie & TV App

A modern streaming-style web app with Electronic Program Guide (EPG) support.

🚀 Features

- 📺 Live TV-style interface
- 🗓️ Electronic Program Guide (EPG)
- 🔥 Firebase integration
- 💡 Clean and responsive UI

🛠️ Tech Stack

- React + TypeScript
- Firebase
- Vite

📂 Project Structure

- "components/" → UI components
- "lib/" → Logic (EPG + Firebase)
- "App.tsx" → Main app

⚙️ Setup

npm install
npm run dev

🌐 Deployment

You can deploy using:

- Vercel
- Netlify
- Firebase Hosting
